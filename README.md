# Task 3 – Superstore Sales Summary

## Description
This project demonstrates SQL basics, including importing datasets, fixing column names, creating summary tables, and exporting results.

## Files
- `task3.db` – SQLite database containing the imported dataset and summary table.
- `sales_summary.csv` – Exported summary of sales by category and subcategory.
- `queries_task3.sql` – All SQL queries used in Task 3.
- `Sample - Superstore.csv` – Original dataset.
- `README.md` – This file.

## Steps Performed
1. Imported Superstore CSV into SQLite.
2. Fixed column names if needed.
3. Created `sales_summary` table using SQL queries.
4. Viewed summary rows to validate data.
5. Exported summary table as CSV.

## How to Use
1. Open `task3.db` in DB Browser for SQLite.
2. Run queries from `queries_task3.sql` to reproduce results.
3. Open `sales_summary.csv` to view exported summary.
